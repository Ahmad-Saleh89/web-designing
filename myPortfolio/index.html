Why Web Development?

It all started in the early 2016 when I enrolled in the CS50 course, introduction to Computer Science (Harvard University) via Edx.org Platform. I learned the basics of C programming language. Later on, I learned the basics of Java via Lynda.com. At that point, I got a decent understanding of how a computer thinks. And I’ve gotten hooked to the programming world ever since.

As 2017 began, I have passionately learned how to build a website using modern HTML, CSS, and JavaScript and in order to keep my work organized; I got myself familiar with the Command Line & GitHub. I have also extended my knowledge by learning Bootstrap, SASS & jQuery.

I still remember the moment when I uploaded my photo to a blank HTML page and linked it to my Facebook page by a click. It was the best feeling ever. Since that moment on, I have been certain that web development will be, for sure, my future career.

This was a fragment about how I started my web development journey. Please feel free to scroll down and take a look on my education sources, skills, certifications earned and more.

I also encourage you to visit About Ahmad page to discover more about me.
