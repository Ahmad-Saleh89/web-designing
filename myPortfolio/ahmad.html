How did I start in web development?

It all started in the early 2016 when I enrolled in the CS50 course, introduction to Computer Science (Harvard University) via Edx.org Platform. I learned the basics of C programming language. Later on, I learned the basics of Java via Lynda.com. At that point, I got a decent understanding of how a computer thinks. And I’ve gotten hooked to the programming world ever since.

As 2017 began, I have passionately learned how to build a website using modern HTML, CSS, and JavaScript. I successfully completed HTML, CSS and JavaScript for Web Developers Course (John Hopkins University) via Coursera platform. After that I kept expanding my experience by taking several courses via Khan Academy, FreeCodeCamp, Lynda.com, Udemy, Udacity, and numerous YouTube courses and tutorials.

And in order to keep my work organized; I got myself familiar with the Command Line & GitHub. I have also extended my knowledge by learning Bootstrap, SASS & jQuery.
